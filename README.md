# Math Worksheet Generator

This project is a clean-room static math worksheet generator for GitHub Pages.

Current implementation stage:

- S4 expression model and generator skeleton

Included in this stage:

- core constants
- default V1 config
- schema-shape documentation helpers
- config validation scaffold
- mixed-pattern planning scaffold
- integer `NumberValue` model
- tree-based integer expression evaluation
- deterministic random helpers
- single-pattern expression generation skeleton
- basic Node test coverage

Not included in this stage:

- worksheet UI
- worksheet rendering
- print CSS
- full mixed worksheet generation
- curriculum datasets

Run tests:

```bash
npm test
```
