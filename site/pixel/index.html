Beta 使用範圍

目前可用功能與已知限制

目前可用

  • 使用目前公開的 Batch A 單元與通過驗證的知識點。
  • 支援單元出題、單一知識點與同單元知識點混合。
  • 知識點有多種題目形式時，可選擇計算題、應用題或兩者混合。
  • 支援題數、排序、答案頁、版面設定、HTML 預覽與列印。

已知限制

  • 跨單元知識點混合尚未開放。
  • 尚未通過 QA、隱藏或未發布的知識點不會出現在選單。
  • 重新整理頁面後不保留設定、考卷或學生作答紀錄。
  • 未列入目前公開 Registry 的單元與題型無法由 Pixel UI 選取。

Pixel UI 是公開 Beta 介面；需要最穩定操作時，可使用上方「回到標準版」。
